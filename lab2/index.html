Тренинг по JavaScript компании NonExistentSoft
Данный тренинг посвящён актуальным подходам, технологиям и инструментальным средствам разработки на JavaScript.
Посещая занятия данного курса, Вы:
  • узнаете о последних тенденциях и трендах в JavaScript
  • познакомитесь с HTML, CSS и HTTP
  • научитесь выполнять юнит-тестирование клиентского кода
  • познакомитесь с архитектурой приложений и структурой типовых проектов
  • получите представление об инструментах современного Javascript-разработчика.
Спектр рассматриваемых тем включает в себя как основы HTML, CSS и JavaScript, так и практическое углубленное изучение популярных веб-технологий.
